The `NumForm` feature applies to [numerals](_xmf/pos/NUM). 

### <a name="Word">`Word`</a>: number expressed as a word

#### Examples

* _ართი (ɑrti)_ 'one', _ჟირი (ʒiri)_ 'two', _სუმი (sumi)_ 'three' etc.

### <a name="Digit">`Digit`</a>: number expressed using digits

#### Examples

* _1, 2, 3_

### <a name="Combi">`Combi`</a>: digits combined with a suffix

#### Examples

* _1-ი (1-i)_ '1st', _მე-11-ე (mɛ-11-ɛ)_ '11th'

### <a name="Roman">`Roman`</a>: Roman numeral

#### Examples

* _I, II, III_
