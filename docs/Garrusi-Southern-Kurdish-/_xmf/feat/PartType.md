The `PartType` feature typically applies to [particles](_xmf/pos/PART), [pronouns](_xmf/pos/PRON), [adpositions](_xmf/pos/ADP), [nouns](_xmf/pos/NOUN), [proper nouns](_xmf/pos/PROPN) [adverbs](_xmf/pos/ADV), [adjectives](_xmf/pos/ADJ), [numerals](_xmf/pos/NUM).

### <a name="Int">`Int`</a>: interrogative particle

#### Examples

* _-ო (-ɔ)_ '?' etc.

### <a name="Ta">`Ta`</a>: affirmative particle

#### Examples

* _ქო- (kɔ-)_ 'yes' etc.

### <a name="Inf">`Inf`</a>: infinitive particle

#### Examples

* -მე (mɛ)_ 'any' etc.

### <a name="Neg">`Neg`</a>: negative particle

#### Examples

* _ვა (va)_ 'not' etc.

### <a name="Emp">`Emp`</a>: emphasis particle

#### Examples

* _ხოლო (xɔlɔ)_ 'too' etc.

### <a name="Foc">`Foc`</a>: focus particle

#### Examples

* _-ც (-t͡s)_ 'and' etc.
