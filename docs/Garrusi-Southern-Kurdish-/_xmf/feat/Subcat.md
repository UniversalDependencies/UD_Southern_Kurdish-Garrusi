The `Subcat` feature applies to [VERB](_xmf/pos/VERB). Megrelian verb can be intransitive, indirect transitive and transitive.

### <a name="Intr">`Intr`</a>: intransitive verb

A verb that does not take arguments other than the subject.

#### Examples

* _ქეიფენს (kɛipɛns)_ 'he/she/it is feasting' etc.

### <a name="Indir">`Indir`</a>: indirect transitive verb

A verb that does not require a direct object but it requires an oblique argument.

#### Examples

* _ბარჯღალანს (bɑrd͡ʒɣɑlɑns)_ 'he/she/it widens' etc.

### <a name="Tran">`Tran`</a>: transitive verb

A verb that takes a direct object as argument (in addition to the subject). 

#### Examples

* _აღორენს (ɑɣɔrɛns)_ 'he/she is lying' etc.
