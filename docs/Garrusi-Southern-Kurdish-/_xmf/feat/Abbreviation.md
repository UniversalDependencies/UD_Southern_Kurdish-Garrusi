Boolean feature of [NOUN](_xmf/pos/NOUN), [PROPN](_xmf/pos/PROPN) or [ADV](_xmf/pos/ADV). It tells whether the common noun, proper noun or adverb are abbreviations.

### <a name="Yes">`Yes`</a>: it is abbreviation


#### Examples

common noun: _გრ. (ɡr.)_ 'gram' etc.
proper noun: _აშშ (ɑʃʃ)_ 'United State of America' etc.
