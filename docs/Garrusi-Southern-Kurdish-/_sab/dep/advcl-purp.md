The dependent of an `advcl:purp` relation is a purposive adverbial clause.


Here is an example from **UD_Bokota-ChibErgIS**:

~~~ sdparse
skira hue para u dodade. \n (I) find some skira to make the house.
advcl:purp (hue, dodade)
~~~
