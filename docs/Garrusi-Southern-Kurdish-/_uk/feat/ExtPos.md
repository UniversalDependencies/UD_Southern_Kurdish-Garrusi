This feature indicates the effective UPOS of an expression, it can differ from the UPOS of the head word.

In Ukrainian, only heads of a [`fixed`](https://universaldependencies.org/en/dep/fixed.html) relation 
employ the feature.

### <a name="ADP">`ADP`</a>: adposition-like expression

#### Examples

* _<b>згідно з</b> програмою_ "according to the programme" (_згідно_ = `ADV`)
* _<b>починаючи з</b> травня_ "starting from May" (_починаючи_ = `VERB`)
* _<b>під час</b> зустрічі_ "during the meeting" (_під_ = `ADP`)

### <a name="ADV">`ADV`</a>: adverb-like expression

#### Examples

* _захищати <b>в першу чергу</b> права людини_ "protect human rights above all" (_в_ = `ADP`)
* _напруга спаде <b>сама собою</b>_ "the tension will ease by itself" (_сама_ = `PRON`)

### <a name="ADJ">`ADJ`</a>: adjective-like expression

#### Examples

* _<b>так звані</b> друзі_ "so-called friends" (_так_ = `ADV`)

### <a name="CCONJ">`CCONJ`</a>: coordinating conjunction-like expression

#### Examples

* _відповідь у тексті, <b>а саме</b> в першому розділі_ "the answer is in the text, namely in the first section" (_а_ = `CCONJ`)

### <a name="SCONJ">`SCONJ`</a>: subordinator-like expression

#### Examples

* _Хочу прокоментувати, <b>тому що</b> це важливо_ "I want to comment because it is important" (_тому_ = `ADV`)

### <a name="PRON">`PRON`</a>: pronoun-like expression

#### Examples

* _дивитися <b>одне одному</b> в очі_ "look into each other's eyes" (_одне_ = `DET`)
* _<b>де завгодно</b>, тільки не тут_ "anywhere but here" (_де_ = `ADV`)

### <a name="DET">`DET`</a>: determiner-like expression

#### Examples

* _<b>свого роду</b> протест_ "a kind of protest" (_свого_ = `DET`)
* _<b>Що за</b> профан це написав?_ "What kind of amateur wrote this?" (_Що_ = `PRON`)

### <a name="INTJ">`INTJ`</a>: interjection-like expression

#### Examples

* _<b>Будь ласка</b>, проходьте!_ "Please come in!" (_будь_ = `VERB`)
* _<b>Слава Богу</b>, ви тепер зрозуміли!_ "Thank God, you understand now!" (_Слава_ = `NOUN`)

### <a name="PART">`PART`</a>: particle-like expression

#### Examples

* _<b>Чи не</b> найбільший у світі_ "perhaps the largest in the world" (_чи_ = `PART`)
