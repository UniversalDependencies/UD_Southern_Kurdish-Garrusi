This feature indicates, for the head of a [fixed expression](https://universaldependencies.org/u/dep/fixed.html), 
the effective UPOS of the full expression when it differs from the UPOS of the word itself.
Occasionally, `ExtPos` is used even if `ExtPos` = `upos`.

`ExtPos` is used in the [SUD](https://surfacesyntacticud.github.io/) framework.
It is kept in FEATS in the UD conversion.

### <a name="ADJ">`ADJ`</a>: adjective-like expression

#### Examples

* _<b>naː íri</b>_ (_naː_ = `NOUN`) "pseudo"

### <a name="ADP">`ADP`</a>: adposition-like expression

#### Examples

* _<b>tu ɗan</b> (_tu_ = `SCONJ`) "like"

### <a name="ADV">`ADV`</a>: adverb-like expression

#### Examples

* _<b>kàmán mìsáːli</b>_ (_kàmán_ = `ADP`) "for example"

### <a name="AUX">`AUX`</a>: auxiliary-like expression

#### Examples

* _<b>yi nə</b>_ "be"

### <a name="SCONJ">`SCONJ`</a>: subordinator-like expression

#### Examples

* _<b>séː deː</b>_ (_séː_ = `ADP`) "except"


### <a name="INTJ">`INTJ`</a>: interjection-like expression

#### Examples

* _<b>tá tu ɓâ</b>_ (_tá_ = `ADV`) "You don't say!"


### <a name="NOUN">`NOUN`</a>: noun-like expression

#### Examples

* _<b>naː kámʃi</b>_  "his friend"

### <a name="PART">`PART`</a>: particule-like expression

#### Examples

* _<b>yâːn tu</b>_  (_yâːn_ = PRON) "that is to say"

### <a name="PRON">`PRON`</a>: pronoun-like expression

#### Examples

* _<b>yádda ɗan</b>_  (_yádda_ = ADV) "how"

### <a name="VERB">`VERB`</a>: verb-like expression

#### Examples

* _<b>fí kòːkàrí</b>_ "try"
