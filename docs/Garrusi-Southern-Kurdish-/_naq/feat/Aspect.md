In Khoekhoe, `Aspect` is a feature of [auxiliaries](naq-pos/AUX_). Aspect is a feature that specifies duration of the action in time, whether the action has been completed or not.

### <a name="Imp">`Imp`</a>: imperfective aspect

The action took / takes / will take some time span and there is no information whether and when it was / will be completed.

#### Examples

* _ǁîs ge ariba <b>ra</b> ǃao._ "She is afraid of the dog."

* _Sas ge ǂhanu tama hâ khoeb ǀkha <b>gere</b> ǀhōgu_. "You were dating the wrong guy."

### <a name="Perf">`Perf`</a>: perfective aspect

The action has been / will have been completed.

#### Examples

* _ǁîb ge ǁîna ǃâu <b>hâ</b>._ "He has waited for them."

* _ǁÎba nē daoba u hā tama <b>hâ</b>?._ "Hasn't he come up this way?"

### <a name="Punct">`Punct`</a>: punctual aspect

The action took / takes / will take place in a clearly bounded short period of time.

#### Examples

* _Tita ge xrat dīsisa ge <b>a</b> ǂnûǃgau._ "I failled grade ten."

* _Funi ge ge ǂgâ oms ǃnâ o ǁîsa <b>a</b> ǂoa._ "Fun entered the house, then she came out."
