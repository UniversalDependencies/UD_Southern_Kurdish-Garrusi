In Khoekhoe, `Deixis` is a feature of [determiners](naq-pos/DET), [pronouns](naq-pos/PRON), and [adverbs](naq-pos/ADV).

### <a name="Prox">`Prox`</a>: proximate

The entity is close to the reference point (e.g., to the speaker).

#### Examples

* _nē_ "this"

* _nē-i_ "this one"

* _nēti_ "in this way, like this"

### <a name="Remt">`Remt`</a>: distal

The entity is far away from the reference point (e.g., from the speaker).

#### Examples

* _ǁnā_ "that"

* _ǁnā-i_ "that one"

* _ǁnāti_ "in that way, like that"

### <a name="Contr">`Contr`</a>: contrastive distal

The entity is far away from the reference point (e.g., from the speaker) and is contrasted to another distal entity.

#### Examples

* _nau_ "that (other)"

* _nau-i_ "that (other) one"

* _nauti_ "in that (other) way"
