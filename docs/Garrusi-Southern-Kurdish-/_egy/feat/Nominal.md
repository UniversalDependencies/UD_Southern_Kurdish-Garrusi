This feature signals nominalization of a verb. If a verb is used in a position where a noun is expected, it keeps the [VERB]() UPOS tag but it will get an additional feature `Nominal=Yes`.

### <a name="Yes">`Yes`</a>: nominalized usage of a verb

#### Example

* _gm ⸗k <b>ḫśf</b> ⸗k i҆m_ “you may find him <b>whom you will meet</b> there” (lit. find.SBJV =2SG.M <b>meet.REL.FUT.M.SG</b> =2SG.M there)
