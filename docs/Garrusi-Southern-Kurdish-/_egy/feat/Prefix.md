### <a name="Yodh">`Yodh`</a>: has the prothetic prefix

Some verb forms are written with an i҆ prefix (also called prothetic) in Old Egyptian, for example the following relative form:

#### Example

* _tꜣ pf <b>i҆:šm(.w)</b> Wni҆ś i҆m ⸗f_ “That land in which Unas walks (...)” (lit. land-m.sg dem:m.sg walk:rel.pres-m.sg Unas-kn in: prep =3sg.m)
